Observable

public class Observable<T>

Observable class. Wrapper of an observable value of generic type T or an error. Observable chains must be retained from the end, as an observable child retains its parent, but the parent does not retains it child (this differs from the Future).

Therefore, this is a good usage of an observable: class MyViewController { let obs : Observable func viewDidAppear(_ eventHub: Observable.Hub) { super.viewDidAppear() // The instance must be retained, otherwise the then closure won’t be called obs = eventHub.subscribe().then { value in print(value: \(value)) // Update UI with value }.fail { error in // Update UI with error } } }
